# Entregable 3 - Desarrollo en Java

Este repositorio contiene el tercer entregable práctico desarrollado para el módulo de **Programación**. 

# 🚀 Descripción del Proyecto

Es un ejercicio para prácticar sobre Arrays en Java, y mi yo friki ha puesto a los alumnos de la academia de la hoja :D
## 🛠️ Tecnologías y Herramientas Utilizadas

* **Lenguaje:** Java (JavaSE-25 / compatible con versiones LTS como Java 17 y 21)
* **IDE:** Eclipse IDE para desarrolladores Java Enterprise y Web
* **Control de Versiones:** Git y GitHub para la gestión del historial y despliegue

## 🧠 Conceptos de Programación Aplicados

Durante el desarrollo de este entregable se han puesto en práctica los siguientes pilares:
* **Estructuras de Datos:** Uso y manipulación de [arrays / colecciones / flujos de datos].
* **Estructura Modular:** Organización limpia del código fuente en paquetes (`src/aplicacion`), separando la lógica principal de la ejecución.
* **Buenas Prácticas:** Código limpio, nombrado autoexplicativo de variables y métodos, y optimización para evitar advertencias (*warnings*) en el entorno de desarrollo.

## 📁 Estructura del Repositorio

* `src/`: Contiene todo el código fuente organizado por paquetes.
    * `aplicacion/`: Clases principales con la resolución de los ejercicios lógicos.
* `.gitignore`: Configuración para mantener el repositorio limpio de archivos temporales de compilación (como la carpeta `bin/`).
